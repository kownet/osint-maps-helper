Bounding Box Helper

It is a simple web project which aim is to help get coordinates of the selected place.

To use it we have to draw a rectangle on the map and next in the top right corner we will see a bounding box coordinates.

With this kind of data we can for example:

  1. Use Twitter Streaming API to catch tweets from specific geographical location.
  2. Use Instagram API to catch photos from specific geographical location.
  3. Use YouTube API to catch movies from specific geographical location.
